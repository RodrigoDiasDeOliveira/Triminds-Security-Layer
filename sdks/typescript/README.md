# SDK TypeScript

SDK TypeScript para consumir o Triminds Security Layer em aplicações web e servidores Node.js.

## Objetivo

- abstrair chamadas ao Gateway de Segurança
- gerenciar tokens JWT/OAuth2
- facilitar integração com o Admin Console e APIs

## Estrutura prevista

- `src/client.ts`
- `src/auth.ts`
- `src/types.ts`
- `src/index.ts`
