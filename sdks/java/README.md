# SDK Java

SDK de referência Java para integração com o Triminds Security Layer.

## Objetivo

Fornecer clientes para:

- autenticação de serviços com OAuth2/JWT
- validação de tokens
- chamadas de APIs seguras
- interoperabilidade com Spring Boot

- criar exemplos de uso para `AuthClient` e `AuditClient`
