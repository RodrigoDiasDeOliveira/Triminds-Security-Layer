# SDK Go

SDK Go para integração com o Triminds Security Layer em microserviços e agentes de infraestrutura.

## Objetivo

- consumir APIs seguras com autenticação JWT/OAuth2
- oferecer utilitários de token e validação de políticas
- suportar chamadas de auditoria e eventos

## Estrutura prevista

- `client/`
- `models/`
- `auth/`
- `utils/`
