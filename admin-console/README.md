# Admin Console

Console de administração do Triminds Security Layer.

## Tecnologias

- React
- Vite
- TypeScript
- Triminds UI
- React Router

## Estrutura do scaffold

- `package.json` - scripts e dependências do frontend
- `vite.config.ts` - configuração do Vite
- `src/main.tsx` - ponto de entrada
- `src/App.tsx` - roteamento principal
- `src/pages/` - páginas do console
- `src/index.css` - estilos base

## Objetivo

Oferecer uma interface para:

- gerenciar usuários e papéis
- configurar políticas de acesso e regras dinâmicas
- visualizar eventos de risco e alerts
- revisar trilhas de auditoria

## Como iniciar

1. `cd admin-console`
2. `npm install`
3. `npm run dev`

> O scaffold já está pronto para evoluir com componentes do Triminds UI.
